Website Performance Overview (May 13 - June 9, 2022)

Key Metrics

Users (99K): Total unique visitors.

New Users (86K): First-time visitors.

Average Engagement Time (1m 37s): Average session duration.

Example: TikTok engagement rate for influencer content is around 18% (Note: This is an example and not directly comparable).

Total Revenue ($234K): Total revenue generated.

New User Acquisition Channels

  • Direct: Users who directly type the website address or use a bookmark.
  • Organic Search: Users who find the website through search engines (SEO is crucial).
  • Referral: Users who arrive from links on other websites.

Example: Credit Karma's business model relies on referrals from embedded offers for credit cards, insurance, etc.

Understanding User vs. New User Counts

The total "Users" (99K) includes both new and returning visitors, while "New Users" (86K) only counts first-time visitors. The difference highlights the number of returning users.

Mobile User Identification

While a significant portion of new users may be on mobile devices, it's important not to assume *all* are. Users access websites from various devices.

To accurately determine the number of mobile users, segment data within Google Analytics by "Device Category" (Mobile, Desktop, Tablet). This is essential for mobile optimization.

Example: Personalized ads, like those for Caudalie skincare, appear across devices, indicating varied access points.
